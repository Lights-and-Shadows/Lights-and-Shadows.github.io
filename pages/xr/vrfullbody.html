VR Full Body

Experimental full body application with various abilities

What is VR Full Body?

A highly experimental test application that uses Unity's XR Interaction Toolkit package to allow for a full body experience within virtual reality. As of the time of this page's publication, it's still very much a work-in-progress project; quite a number of possible features are planned, which include:

  • Climbing various objects and obstacles in the player's environment
  • One-handed and two-handed weapon operation
  • Various item manipulation options (eg. loading of magazines for weapons, interacting with objects)

This project is heavily inspired by BONEWORKS and Hotdogs, Horseshoes, and Hand Grenades, two excellent VR titles that I highly recommend checking out if you get the chance. I've always wanted a sandbox-style game that simply lets you mess around with the environment (like the two aforementioned games).
